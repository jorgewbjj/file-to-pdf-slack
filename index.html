Upload File to Convert to PDF and Send to Slack

This website uses a GitHub Actions workflow to convert uploaded files to PDF and send them to a Slack channel. To use it:

  1. Ensure the SLACK_BOT_TOKEN and SLACK_CHANNEL_ID secrets are set in the repositoryâ€™s settings.
  2. Upload a file to the uploads directory of the repository. You can do this by navigating to the uploads folder, clicking Add file > Upload files, selecting your file and committing the change to the main branch.
  3. The GitHub Actions workflow will run automatically, create a PDF summary of your file, and post it to the specified Slack channel.

Supported file types include text files and other documents. Binary files will include only metadata in the PDF summary.
